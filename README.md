# Saberes Científicos Blog

Repositorio listo para desplegar en **Vercel** con **Next.js**.

## Requisitos

- Node.js 18.18 o superior
- npm

## Ejecutar en local

```bash
npm install
npm run dev
```

Abre `http://localhost:3000`.

## Desplegar en Vercel

1. Sube este repositorio a GitHub.
2. Entra a Vercel.
3. Importa el repositorio.
4. Vercel detectará automáticamente **Next.js**.
5. Haz clic en **Deploy**.

## Estructura

- `app/` → rutas y layout principal
- `components/SaberesCientificosBlog.jsx` → tu componente del blog
- `app/globals.css` → estilos base globales
- `vercel.json` → configuración simple para Vercel
